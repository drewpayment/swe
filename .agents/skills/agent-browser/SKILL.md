# agent-browser

`agent-browser` is an npm package for browser automation, designed for navigating websites and executing UI-focused tasks programmatically.

## Package Details

- **Package name**: `agent-browser`
- **Registry**: npm (`npm install agent-browser` / `bun add agent-browser`)
- **Purpose**: Provides a programmable browser agent that can visit URLs, interact with page elements, extract content, and perform multi-step UI workflows.

## When This Applies

Any time "agent-browser" is referenced in conversation, code, or configuration, it refers to this npm package — not a generic concept of a browser-based agent.

## Typical Use Cases

- Automating browser-based workflows (form filling, navigation, clicking)
- Scraping or extracting structured data from web pages
- Testing UI flows end-to-end with a real browser
- Performing authenticated web interactions on behalf of a user
- Executing multi-step tasks that require visual/DOM context

## Integration Notes

- Install via npm/bun into a Node.js or Bun project
- Typically used server-side or in worker processes, not in client-side browser code
- Can be combined with other tools and agents to handle tasks that require web interaction
